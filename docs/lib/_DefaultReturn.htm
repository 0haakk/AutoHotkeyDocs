#DefaultReturn [v2.1-alpha.2+]

Sets the default return to either "" or unset.

Warning: This directive is experimental and may be replaced by a different mechanism in a future release.

#DefaultReturn unset
#DefaultReturn ""

Remarks

#DefaultReturn unset causes any subsequent parameterless return statements or function's closing brace to act as return unset instead of return "".

#DefaultReturn "" restores the original default behavior.

This directive may be replaced by a different mechanism for enabling new language features. For example, importing the v2.1 "module" may enable this and other changes, or sub-modules could exist to allow new behavior to be enabled more selectively.

Related

Unset

Examples

Changes the default return for only f1 and f2.

#DefaultReturn unset
f1() {
    return
}
f2() {
}
#DefaultReturn ""
f3() {
}
MsgBox "f1: " (f1() ?? "unset") "`n"
    .  "f2: " (f2() ?? "unset") "`n"
    .  "f3: " Type(f3())
